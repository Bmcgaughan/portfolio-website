What do I Watch?

Site built with React to help answer the question - what do I watch tonight?
It leverages The Movie Database API(s) to get the top trending shows each week and also provide recomendations based on what you 'favorite' and if you have favorites in common with other users. Click a show image to get a plot summary and search for similar shows.

Back End built using Node.js, Express, Passport, Mongoose and MongoDB and hosted on Heroku check out the back-end on github

Front End built with React, Redux, and Swiper.js and hosted on Netlify

Meet-App

This is a Serverless Progressive Web App built in React. It uses AWS Lambda, OAuth2 through Google, and the Google Calendar API. It shows all the events from a public Google Calendar and allows the user to filter for specific cities.

There are also some visualizations from Recharts

This is a work in progress. Eventually I will allow filtering by time/date.
